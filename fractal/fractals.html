Koch Curve

Original

<----- koch_curve(t, 4, 20)

Modified 

 
        def koch_curve_2(t, depth, size):
          if depth == 0:
          t.fd(size)
          else:
            for i in [[85,1.3], [190,1], [85,1], [0,1]]:
              koch_curve_2(t, depth - 1, size // (3*(i[1])))
              t.left(i[0])
              
        This modification makes the koch curve
        have variations in the measurements for
        length and angles.
